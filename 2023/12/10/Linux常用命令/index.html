Linux系统常用命令
  1. ls - 列出目录
  2. pwd - 打印当前目录
  3. cd - 进入指定目录
  4. mkdir - 创建目录
  5. mv - 移动或重命名文件
  6. cp - 复制文件
  7. rm - 删除文件或目录
  8. touch - 创建空文件
  9. ln - 创建指向其他文件的快捷方式
  10. cat - 在终端上展示文件内容
  11. clear - 清除终端显示
  12. echo - 打印命令后的文本
  13. less - 在终端上显示分页输出
  14. man - 访问所有Linux命令的手册
  15. uname - 获取操作系统的基本信息
  16. whoami - 获取当前用户名
  17. tar - 解压缩文件
  18. grep - 在输出中搜索字符串
  19. head - 从顶部返回指定的行数
  20. tail - 从底部返回指定的行数
  21. diff - 找出两个文件之间的差异
  22. cmp - 检查两个文件是否相同
  23. comm - 结合diff和cmp的功能
  24. sort - 在输出时对文件内容进行排序
  25. export - 导出Linux环境变量
  26. zip - 打成压缩包
  27. unzip - 解压缩
  28. ssh - SecureSheel命令
  29. service - 用于启动和停止服务
  30. ps - 显示活动进程
  31. kill and killall - 通过进程ID或进程名终止活动进程
  32. df - 显示磁盘文件系统信息
  33. mount - 挂载文件系统
  34. chmod - 修改文件权限
  35. chown - 授予文件或文件夹的所有权
  36. ifconfig - 显示网络接口和IP地址
  37. traceroute - 跟踪到达目的地的所有网络跳数
  38. wget - 直接从互联网下载文件
  39. ufw - 防火墙命令
  40. iptables - 用于与所有其他防火墙实用程序交互的基本防火墙
  41. apt, pacman, yum, rpm - 包管理器
  42. sudo - 升级权限
  43. cal - 查看命令行日历
  44. alias - 为经常使用的命令创建自定义快捷方式
  45. dd - 主要用于创建可指导的USB闪存盘
  46. whereis - 找到命令的二进制文件、源代码和手册页
  47. whatis - 查找命令的用途
  48. top - 查看活动进程及其系统使用情况
  49. useradd and usermod - 添加新用户或更改现有用户数据
  50. passwd - 为现有用户创建或更新密码
